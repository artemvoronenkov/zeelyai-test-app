# Zeely AI Test App

[![Deploy To GitHub Pages](https://github.com/artemvoronenkov/zeelyai-test-app/actions/workflows/deploy-pages.yml/badge.svg)](https://github.com/artemvoronenkov/zeelyai-test-app/actions/workflows/deploy-pages.yml)

Live demo: https://artemvoronenkov.github.io/zeelyai-test-app/

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployment is automated via GitHub Actions on every push to `main`:

- Workflow: `.github/workflows/deploy-pages.yml`
- Target: GitHub Pages
